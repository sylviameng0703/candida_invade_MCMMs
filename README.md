# candida_invade_MCMMs
This repository contains code and data to reproduce analyses for the manuscript 'Integrative metagenomics and metabolic models reveal mechanistic associations between vaginal dysbiosis and Candida albicans engraftment risk'
- Data used to process species level composition can be found in the data folder, tables exceeding 25 MB can be founder here: .
- MCMMs we build can be found in the models folder.
- Scripts used to generate predicted growth rates and fluxes can be found in the scripts folder.
- Jupyter notebooks used for data analysis and the production of figures in the manuscript can be found in the scripts/notebooks folder.
